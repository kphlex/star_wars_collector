Project description:
Users can collect items from the Star Wars world. Including people, starships, and planets.

Utilizing SWAPI Star Wars API 
https://swapi.dev/

MVP: 
- Login / Registration with validations. 
- Show all items the user has collected. 
- Roll random items (3 per roll).
- View user's profile.
- Allow users to comment on user's profile.
- Allow users to delete items in their collection. 
- Click on item to view details. 
- View all users.

Bonus Features:
- Time gate rolls. ( 2 rolls per 18hrs. )
- Display time remaining...
- Deploy application.
